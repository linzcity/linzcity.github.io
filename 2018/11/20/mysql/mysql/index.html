Linux上的mysql

阿里云 + mysql

  1. 买了服务器后直接使用镜像市场，选择java环境，重置系统镜像（如：jdk + tomcat + mysql）
  2. 云服务器远程连接：winscp
  3. 云数据库远程连接：navicat、或者windows的cmd命令行
  4. 首先设置云数据库白名单：IP白名单设置为0.0.0.0/0代表允许所有地址访问，设置为127.0.0.1代表禁止所有地址访问
  5. 设置0.0.0.0/0很危险：只设置某台机器本地远程连接的话，cmd命令查询到的ip有问题。

安全远程连接云数据库mysql：阿里云官网方案


附：在windows系统上操作（我的为例）:

  1. 在mysql安装目录bin目录下，运行cmd窗口（如果配置了全局环境就不用指定bin目录）
  2. 输入连接命令：mysql -hxxx.mysql.rds.aliyuncs.com -u账户 -p (windows下，密码不直接跟在p后面，否则不给操作，端口也不用写了。敲完回车输密码)
  3. 连接上后，命令：show processlist
  4. 把列出来的ip加到白名单即可

远程连接云服务器上的Mysql数据库

  1. 登录云服务器
  2. 进入mysql安装目录
  3. 登录root账户
  4. 授权
    4.1. 你想用户使用密码从任何主机连接到mysql服务器的话:具体ip改为%
    4.2. 如果你想允许指定用户从ip为192.168.90.101的主机连接到mysql服务器，并使用密码作为密码 : GRANT ALL PRIVILEGES ON . TO ‘root’@‘192.138.88.222’ IDENTIFIED BY ‘123’ WITH GRANT OPTION;
  5. 同时打开云服务器的3306端口访问权限。这样navicat就能远程访问数据库了。

补充：修改访问ip白名单，可以直接用命令，也可以先设置%，再利用navicat登录，然后在mysql.user下将对应的用户名下的host字段改为指定ip。
